# SEO + Metadata Rules

## Global
- Every page has unique:
  - title
  - meta description
  - canonical (if needed)
- Open Graph and Twitter card metadata set globally, override per page if needed.

## Copy constraints
- Avoid “stock picks” keywords
- Prefer: “stock screener”, “SEC filings”, “fundamental screening”, “unbiased filters”

## Structured content
- Updates posts should have:
  - title
  - date
  - summary
  - slug
